# Cloud Cost Optimizer

**Cloud Cost Optimizer** is an intelligent agent designed to help Google Cloud users **reduce costs and optimize their cloud resources**. It provides recommendations on compute, storage, and other cloud resources, helping teams save money while maintaining performance.

---

## Features

- **Compute Optimization**: Identify underutilized VMs and suggest rightsizing.
- **Discount Recommendations**: Recommend committed use contracts and preemptible VM usage.
- **Storage Optimization**: Suggest lifecycle policies to reduce storage costs.
- **Autoscaling & Monitoring**: Detect idle resources and optimize scaling strategies.
- **Interactive Chatbot**: Ask questions and receive actionable recommendations in real time.
